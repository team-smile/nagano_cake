# ECサイト　長野ケーキ
## DmmWebCamp　チーム開発（4名） 　

## 概要
### ケーキを購入できるECサイト

## 主な機能
 - ユーザー登録
 -　ログイン・ログアウト
 - カート
 - 商品一覧
 - 商品検索
 - マイページ
 - 購入履歴一覧・削除
 - パスワード再設定

## 開発期間
2020/10/27~2020/11/14(18日)

## 開発環境
- 統合開発環境: AWS cloud9
- 開発言語: Ruby 2.6.3
- フレームワーク: Rails 5.0.7
